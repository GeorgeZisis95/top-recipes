# Sweet Treats Recipe Project

A simple, warm, and responsive recipe website featuring three delicious recipes:

- Peachy Dream Smoothie
- Vanilla Cloud Cupcakes
- Caramel Kisses Brownies

The project is built with **HTML, CSS, and JavaScript**, and designed to be mobile-friendly with a consistent layout and interactive elements like a "Back to Top" button.

---

## Features

- **Responsive design**: Content scales nicely on mobile and desktop screens.
- **Consistent layout**: All sections (Description, Ingredients, Instructions, Nutritional Info) are aligned and centered.
- **Interactive elements**:
  - "Back to Top" button that appears on scroll.
  - Smooth hover effects on navigation links.
- **Recipe images**: Circular and visually appealing.
- **Favicon**: PNG icon added to all pages.
- **Navigation**: Easy access to Home from each recipe page.
